# 자율주행 파트 계획서

**담당 파트: 자율주행 (객체 인식 기반 추적 주행)**

---

## 1. 개요

본 파트는 오토카에 탑재된 카메라와 라이다 센서를 이용하여, **사람을 인식하고 자율적으로 따라 주행**하는 기능을 구현한다. 기존에 고려되었던 트랙 라인 트레이싱 방식은 현장 테스트 결과 강한 역광 환경에서 색상 기반 인식의 정확도를 보장하기 어렵다고 판단하여 채택하지 않았다. 대신 **YOLO 객체 인식을 이용한 사람 추적 주행**을 핵심 기능으로 선정하였다. 이는 라인이나 트랙 같은 고정된 환경 조건에 의존하지 않고, 실외 다양한 장소에서 재현 가능하다는 장점이 있다.

## 2. 목표

- 카메라로 촬영한 영상에서 사람을 실시간으로 인식한다.
- 인식된 사람과의 상대적 위치(좌우 방향)와 거리를 판단하여, 차량이 해당 인물을 일정 거리를 유지하며 따라가도록 조향 및 속도를 제어한다.
- 라이다를 이용해 진행 방향의 장애물과의 거리를 상시 감지하여, 충돌 위험 시 우선적으로 정지하는 안전 기능을 병행한다.

## 3. 시스템 구성

| 구성 요소 | 역할 |
|---|---|
| 정면 카메라 (CSI) | 사람 인식을 위한 영상 입력 |
| YOLO (YOLOv3-tiny, TensorRT 최적화) | 실시간 사람 탐지 및 화면 내 위치/크기 산출 |
| 라이다 | 360도 거리 측정, 충돌 위험 시 최우선 정지 신호 제공 (안전 보조) |
| Jetson Nano | 위 두 센서의 판단 결과를 받아 조향/속도 명령으로 변환, 실행 |
| 오토카 구동부 (`Pilot.AutoCar`) | 실제 조향 및 전후진 제어 |

**역할 분리 원칙**: 카메라(YOLO)는 "무엇을, 어느 방향에서" 인식하는 역할을, 라이다는 "얼마나 가까운지"를 판단하는 안전 담당 역할을 맡아 서로 보완한다. 두 판단이 충돌할 경우, 안전을 위해 라이다의 정지 판단이 항상 우선한다.

## 4. 동작 알고리즘

### 4-1. 사람 인식 및 추적 로직
1. 매 프레임마다 YOLO로 화면 내 모든 객체를 탐지하고, 그중 `person` 클래스만 필터링한다.
2. 여러 명이 인식될 경우, 화면상 박스 면적이 가장 큰(=가장 가까운) 사람 한 명을 추적 대상으로 선정한다.
3. 대상의 화면 내 좌우 위치(중심 좌표)를 화면 중앙과 비교하여 조향값을 계산한다. 대상이 왼쪽에 있으면 좌회전, 오른쪽에 있으면 우회전하여 항상 화면 중앙에 위치하도록 한다.
4. 대상의 박스 크기(화면 대비 면적 비율)를 거리의 근사치로 사용한다.
   - 대상이 작게(멀리) 보이면 전진
   - 적정 거리 범위에 있으면 저속 유지
   - 지나치게 가까우면 정지
5. 대상이 화면에서 사라지면 안전하게 정지한다.

### 4-2. 라이다 기반 안전 정지
- 사람 추적 로직과 독립적으로, 라이다가 전방 근접 장애물을 감지하면 즉시 정지 명령을 우선 적용한다.
- 이는 추적 대상 외의 장애물(벽, 다른 사물 등)과의 충돌을 방지하기 위한 최소한의 안전장치이다.

## 5. 기술적 특징

- **경량 모델 사용**: on-device 환경(Jetson Nano)의 제한된 연산 성능을 고려하여 YOLOv3-tiny를 채택하였다.
- **TensorRT 최적화**: 초기 OpenCV DNN 기반 실행 시 초당 6프레임 수준이었던 처리 속도를, TensorRT 엔진 변환을 통해 초당 30프레임까지 향상시켜(약 5배) 실시간 추적이 가능한 수준을 확보하였다.
- **오프라인 동작**: 별도의 인터넷 연결 없이 Jetson 자체에서 인식·판단·제어가 모두 이루어진다.

## 6. 향후 진행 계획

| 단계 | 내용 |
|---|---|
| 완료 | YOLO 모델 경량화 및 TensorRT 변환, 실시간 추적 로직 구현 |
| 진행 예정 | 사람 추적 기능 실기(현장) 테스트 및 조향/속도 파라미터 보정 |
| 진행 예정 | 라이다 실제 연결 및 안전 정지 기능 통합 테스트 |
| 진행 예정 | 사람 추적 + 라이다 안전 정지 통합 최종 테스트 |

## 7. 기대 효과

트랙과 같은 고정 환경 없이도 실외 다양한 장소에서 시연이 가능하며, 사람을 인식하고 따라간다는 직관적인 기능으로 시연 시 전달력이 높다. 또한 라이다를 통한 이중 안전장치를 갖추어, 인식 오류나 예상치 못한 장애물 상황에서도 안전하게 정지할 수 있는 구조를 갖춘다.
